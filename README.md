## Local Development

To build and run the curriculum site locally,

1. Download Python [here](https://www.python.org/downloads/)
2. From a command line, go to your SIDEQUESTS folder and run

    pip install mkdocs

    pip install mkdocs-material

    cd src
    
    mkdocs build
    
    mkdocs serve
